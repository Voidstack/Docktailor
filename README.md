# \# Projet Java Multi-Fenêtre

# 

# \## Description  

# Application Java avec interface graphique multi-fenêtre permettant de gérer plusieurs vues et interactions utilisateur de façon fluide.

# 

# \## Fonctionnalités  

# \- Gestion de plusieurs fenêtres et dialogues  

# \- Architecture modulaire avec controllers dédiés  

# \- Utilisation de JavaFX et design MVC  

# \- Système de navigation entre fenêtres intégré

# 

# \## Installation  

# 1\. Cloner le dépôt  

# 2\. Importer le projet dans IntelliJ IDEA  

# 3\. Compiler et lancer `MainApp.java`

# 

# \## Technologies  

# \- Java 17+  

# \- JavaFX  

# \- Maven/Gradle (selon projet)  

# \- Reflections (pour découverte dynamique de classes)

# 

# \## Usage  

# Lancer l’application et naviguer entre les fenêtres via le menu principal ou boutons d’action.
